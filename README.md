# Tracker

Personal habit tracker — plain HTML/CSS/JS, no build step, no dependencies. Data lives in `localStorage`. Installable as a PWA (offline-capable, add to home screen).

**Sections:** points economy with manual confirmation and rank ladder · gym program with per-muscle lift-based ranks · calorie/macro logging with a meal database · weekly indulgence budget · user-defined goals · rewards shop · stats + full JSON backup/restore · a three-character scripted narrator (Stanley Parable-style).

**Run locally:** serve the repo root with any static server and open `index.html` (service worker needs http, not `file://`). Deployed on GitHub Pages.
